# Lox-ts-bytecode

Implementation of [Lox](https://github.com/munificent/craftinginterpreters) in Typescript for studying purposes and focus on part III of the book.

# Notable differences to cLox

- none so far


# Building

```
$ npm install
```

# Running

REPL:

```
$ npm run-script run
````

From file


```
$ npm run-script run path/to/file
````
